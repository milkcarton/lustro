Versie geschiedenis & bugs

We hopen regelmatig nieuwe versies van Lustro te kunnen uitbrengen met nieuwe opties en minder foutjes. Deze lijst bevat alle aanpassingen opgedeeld per versie.

Versie 1.0 2008.06

  • Lokalisatie: Lustro vertaald in het Nederlands
  • Export knop gedeactiveerd bij exporteren
  • Bewaren van het log bestand
  • Menu export optie gedeactiveerd wanneer de export knop gedeactiveerd is
  • Mooiere CSS layout voor de hCard export
  • Titels bij voor de CSV en tab bestanden
  • Gebruiker laten kiezen waar de export bestanden worden opgeslagen
  • Google export optie activeren wanneer aanmelden gelukt is
  • Geëxporteerde contactpersonen alfabetisch sorteren op achternaam, voornaam en bedrijf
  • Verbeterde gebruikersinterface voor tab navigatie
  • NIB bestanden geconverteerd naar XIB
  • Lustro crasht niet wanneer Sleutelhangertoegang niet wordt toegestaan
  • Bevestiging voor Google: vraag bevestiging voor het overschrijven van de Google Contactpersonen als extra controle

Versie 0.9 2008.06.19

  • Bèta versie, gebruikt om de laatste fouten te vinden

Foutje gevonden?

Heb je een bug gevonden? Iets gevonden dat niet werkt zoals het moet? Controleer eerst of je de laatste nieuwe versie van Mac OS X gebruikt met de nieuwste updates (‘Software-update...’ onder het Apple icoontje in de linker bovenhoek). Contacteer ons (dat kan gerust in het Nederlands) en vertel ons hoe we het probleem kunnen reproduceren zodat we een oplossing kunnen zoeken. Het is belangrijk dat je ons kan vertellen welke stappen je uitvoert voordat het fout gaat zodat wij hetzelfde gedrag kunnen reproduceren (anders is het voor ons nogal moeilijk om te weten of de probleem opgelost is). Vertel ons eventueel ook welk systeem je gebruikt, dit kan nuttige achtergrond informatie zijn.

Voor de ‘geeks’ onder ons: Lustro is open source dus als je wil kan je de code downloaden (Lustro SVN repository), het probleem oplossen en ons de patch doorsturen. Verwacht eeuwige roem en dankbaarheid!
