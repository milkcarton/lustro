Found a bug?

Found a bug? Do you think you found something that doesn't work while it should? Make sure you upgraded your Mac OS X with the latest security fixes (‘Software Update...’ under the Apple icon in the right top corner). Contact us and tell us how we can reproduce the problem and we'll have a look at it. It's important you can tell us how to reproduce the problem (or it will be hard for us to know if it's fixed) and tell us your system details.

For the geeky people out there: Lustro is open source so if you feel up to it go and download the source code (Lustro SVN repository), fix the problem and send us the patch. Eternal fame will be yours!
